LAMP Stack

es un conjunto de software Open Source
Especificación Descripción
Sistema Operativo LAMP funcionará de forma correcta únicamente en Linux, aunque existen adaptaciones LAMP para utilizar bajo Windows o MacOS.
Servidor Web En LAMP es Apache quien lo ofrece, y su tecnología está tan probada y madura en comparación con Node.JS. Ya lleva más de una década entre nosotros y camino de las dos…
Código LAMP hace uso de PHP, aunque se puede reemplazar por Python o Perl.
Almacenamiento de datos LAMP usa como gestores de base de datos a MySQL o MariaDB, que son bases de datos de tipo relacional.
